📉 Backtesting & Monte Carlo

Historical simulation (2014–2025) + Monte Carlo stress testing. Uses your current portfolio.

Methodology: Most covered-call ETFs launched 2020–2023, so this tool reconstructs historical performance using the underlying index's actual yearly returns (S&P 500, QQQ, Russell 2000) combined with each ETF's covered-call cap factor and yield profile. ETFs with real history (QYLD 2013, XYLD 2013, DIVO 2016, JEPI 2020) use actual returns where available. Monte Carlo runs 1,000 simulations with random annual returns drawn from the historical distribution plus fat-tail shock events (5% chance each year).
